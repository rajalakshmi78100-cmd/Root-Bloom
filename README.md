# 🌿 Root & Bloom

Root & Bloom is a responsive plant and flower e-commerce website built using **HTML** and **Tailwind CSS**. The website provides a clean and modern user interface for users to explore and purchase a variety of plants and flowers.

## 🚀 Features

* Responsive design for desktop, tablet, and mobile devices
* Modern UI built with Tailwind CSS
* Beautiful product showcase section
* Contact form for customer inquiries
* Easy navigation across pages

## 📄 Pages

### 1. Home Page

* Attractive hero section
* Featured plants and flowers
* Navigation bar
* Call-to-action sections

### 2. Products Page

* Product listing grid
* Search bar for filtering products
* Product cards with images, names, and prices
* Buy Now button for each product

### 3. Contact Page

* Contact form
* User-friendly layout
* Fields for customer inquiries and feedback

## 🛠️ Technologies Used

* Tailwind CSS
